> JSON file of ISO 3166-2 subdivisions.
> This repository was forked from https://github.com/olahol/iso-3166-2.json
> It fixes typos

## Install

```bash
$ npm i iso-3166-2.json --save
```

## Layout

```js
{
  [country code]: {
    name: [country name]
    divisions: {
      [subdivision code]: [subdivision name]
    }
  }
}
```

or

```js
{
  ...
  US: {
    name: "United States"
    divisions: {
      ...
      "US-CA": "California"
      ...
    }
  }
  ...
}
```
